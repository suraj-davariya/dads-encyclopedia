# AI Skill: Operations and Utility Scripts

## Context
When creating one-off utility scripts, git operation scripts (like rewriting commits), or any temporary automation tools that are not part of the core application.

## Instructions
1. **Never place utility scripts in the project root.** 
2. Always put these scripts inside a dedicated folder (e.g., `.ops/` or `scripts/`).
3. **Always add this folder to `.gitignore` immediately.** We do not want to pollute the project's commit history with temporary or useless operational files.
4. Clean up temporary files generated by these scripts once the operation is complete.
